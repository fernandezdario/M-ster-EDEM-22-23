## Docker
### ¿Qué es Docker?

Docker es una plataforma abierta para desarrollar, enviar y ejecutar
aplicaciones en contenedores.

Con Docker, se puede gestionar su infraestructura de la misma manera que
gestionamos sus aplicaciones.
